## Before You Begin

Confirm the source course shell, destination course shell, term, course owner, and whether all components should be copied.

## Procedure

1. Open the destination course shell in Brightspace D2L.
2. Select **Course Admin**.
3. Open **Import/Export/Copy Components**.
4. Select **Copy Components from another Org Unit**.
5. Search for and select the source course shell.
6. Choose all components or select individual components for the build.
7. Review the selected components before starting the copy.
8. Start the copy and wait for completion.
9. Review the copy log for warnings or failed items.
10. Spot check content modules, assessments, links, and dates.

## Troubleshooting / Notes

- Copy only required components when the destination shell already has current materials.
- Confirm gradebook settings after any quiz or assignment copy.
- If the copy fails, capture the copy log and submit an Ivanti ticket.
