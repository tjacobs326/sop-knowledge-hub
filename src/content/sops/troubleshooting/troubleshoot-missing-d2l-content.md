## Before You Begin

Collect the course code, content title, learner or instructor role, screenshot, and date the content was expected to appear.

## Procedure

1. Confirm the content exists in the course shell.
2. Check whether the module, topic, quiz, assignment, or discussion is hidden.
3. Review start dates, end dates, release conditions, and group restrictions.
4. Confirm the user is enrolled in the correct role and section.
5. Compare the course shell to the source shell when content was copied.
6. Test the content using an appropriate learner or preview role.
7. Correct visibility, date, or release condition issues when authorized.
8. If the issue cannot be resolved, submit an Ivanti ticket with screenshots and findings.

## Troubleshooting / Notes

- Hidden parent modules can make visible child topics appear missing.
- Release conditions may block only some learners.
- Date restrictions can be easy to miss after a course copy.
