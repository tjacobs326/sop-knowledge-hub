## Before You Begin

Gather the affected user's name, email address, course or section code, relevant dates, and any screenshots that show the issue or request.

## Procedure

1. Open Ivanti and sign in with your organization credentials.
2. Select **New Ticket**.
3. Choose the request category that most closely matches the issue.
4. Enter a clear title that names the system, course, and action needed.
5. Describe the issue or request in complete sentences.
6. Add affected course, user, or system information.
7. Attach screenshots when available.
8. Set the priority based on business impact and urgency.
9. Submit the ticket.
10. Save the ticket number for reference.

## Troubleshooting / Notes

- If the request affects multiple courses, list each course code separately.
- If the issue blocks student access or course launch, mark the priority as high and include the deadline.
- If the user cannot provide screenshots, document the exact error text and where it appears.
